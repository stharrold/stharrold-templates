# Deployment Guide

Documentation for applying stharrold-templates workflow system to target repositories.

## Quick Start

1. **New repository**: Use `initialize_repository.py` (interactive wizard)
2. **Single existing repo**: Manual copy + merge
3. **Multiple existing repos**: Use `apply-workflow-batch.sh`

See `decision-matrix.md` for detailed guidance on method selection.
